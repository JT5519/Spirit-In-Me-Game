# Spirit-In-Me-Game

Click [here](https://jt5519.github.io/Spirit-In-Me-Game/index.html) to play the game on a GitHub static page! (Recomended screen resolution: 1920 x 1080)

This is a reduced size version of the game, to make it hostable on a GitHub static page. Go to the [main repo](https://github.com/JT5519/Spirit-In-Me) for the original game files and project files. 

# Game Details
* Survival/Action horror game
* Play as a hunter of supernatural entities that also has a special ability to leave their physical body as a spirit and beat the shit out of supernatural entities
* 20 minutes of story driven gameplay 
* A challenging enemy AI 
* Significant exploration and mystery components 
* Free roam
